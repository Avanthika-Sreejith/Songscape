# Songscape

Songscape is a responsive music website template developed using HTML, CSS, and JavaScript. The website provides users with an engaging platform to explore music, view artist information, and contact the developers through a responsive interface. It is designed to work seamlessly across desktop, tablet, and mobile devices.

## Live Demo

https://avanthika-sreejith.github.io/Songscape/

## Repository

https://github.com/Avanthika-Sreejith/Songscape

---

## Features

- Responsive navigation bar
- Home page with hero banner
- Music collection page
- Artist details page
- About page
- Contact page
- Audio playback
- JavaScript form validation
- Responsive design for all screen sizes
- Hover effects and animations
- Footer with navigation links

## Form Validation

The contact form validates the following:

- All fields are required.
- Email must be in a valid format.
- Phone number accepts only numeric values.
- Phone number must contain exactly 10 digits.
- Message length is limited according to project requirements.

## Installation

1. Clone the repository

```bash
git clone https://github.com/Avanthika-Sreejith/Songscape.git
```

2. Navigate to the project directory

```bash
cd Songscape
```

3. Open `index.html` in your browser

or

Use the Live Server extension in Visual Studio Code.
